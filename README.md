14. 《我的三体罗辑传》 再次看一遍这种MC的动画片，尽管当初看这个很难接受这种画风，但是越看还是越感动，感谢制作者，感谢大刘。
15. 《我的三体章北海传》无知不是让人毁灭的，傲慢才是。动画越来越制作精良，水滴两千响的时候，再看一遍还是感人。
16. 成龙《警察故事》成龙的电影很多时候都是注重动作设计，而故事性就稍微弱一点。另外其实我最讨厌的就是相互怀疑的一种猜疑链。
17. 《海蒂和爷爷》很不错的儿童故事，感觉童年的确需要一个这么好的朋友。
18. 《扬名立万》好像是万合时代出品的一部电影，很典型的剧本杀作品，就像之前玩《古木吟》的时候，只不过和古木吟不同的是，电影降低了玩剧本杀时的迷茫感，后期有一些隐喻，又有一些反转，仿佛看到了 编剧躲在作品后面在笑。总的来说是一部不错的电影，唯一可惜的就是感觉前面铺垫那么久，但是好像很难共情。
19. 《虎口逃生》，经典的喜剧，虽然我没笑，但是能够从后代发现很多借鉴
20. 《模仿游戏》编剧的某些剧情用在图灵上，看的我很不舒服，为了加强剧情冲突，加了太多东西，太好莱坞以至于我很不喜欢，不明白为什么这也能豆瓣TOP250，作为学计算机的实在是看不下去
21. 《绣春刀》武侠框架下的朝廷总会存在一种违和感，违和感在于仿佛官职与武功高低绑定了，但这种武功高低本身是在武侠框架之内的。所以武侠框架下的电影就算想表达困境下的人，但总会因为武侠而破坏掉那种困境的意境。
22. 文牧野《奇迹笨小孩》可以看到很多我不是药神的影子，以及整部电影想讲现实，但总点到为止，没能给观众更好的体验。总的来说，及格之作，但也只是及格。
23. 《波斯语课》还行，整个概念很有意思，甚至会让人觉得新鲜。但是感觉没给人那么震撼。
24. 《至暗时刻》绝佳的光影体验，给人一种在黑暗中窒息的感觉，从而突然看到光明，政客之间的博弈省略了点，最后做地铁做了艺术加工，但总体来说可以给9分以上的电影。

+ 暂时还没有，等到时候有什么感悟要及时记录下来。
+ 好的作品的输出除了作品中的故事以外，其实有些时候还要输出一些似乎让读者相信的道理，然后打破这些道理，能够让读者得到反转的快感。（仅限于jojo中的感悟）
+ 如果所有自己所做的事情都能够记录下来，那么对于自己来说的成长无疑是巨大的。
+ 疫情让我意识到未雨绸缪的重要性，君子不立于危墙之下，这句话不仅仅是字面意思，还要做好应急准备，以后就算工作了也要做好应急准备。要少花钱，多存钱。人不能异化成资本，但是也不能异化成为资本的走狗。
+ 最近经常睡不到，白天总是靠咖啡续命，从之前的偶尔一杯，到每天早上一杯，到现在早上下午一杯。有时候傍晚还一杯，我感觉我已经离不开咖啡了，这实在是让人悲伤
+ 整个世界太复杂了，没人能看清，再加上最近互联网的裁员，让我觉得有些焦虑。静下心来后，我问自己，自己想要做什么样的人？最近终于有一些行动去做。首先要做的就是努力了解世界，然后做一个技术大佬。
是的，如果作家是自己不能达到的，那么最起码做一个技术的大佬也是可以的，因此开始每天写代码。另外虽然世界复杂，但是努力了解世界，再认真读文学，读经济，以后还要读史书。首先从经济开始，经济最能够发现一个世界是如何运转的。
+ 互联网的产品实在是太烂了，一个个的追求绩效，追求用户留存率，把许多很简单的事情做得越来越复杂，越来越臃肿，国内互联网的游戏也实在是太烂了，所有的核心就是逼氪，真的是恶心他妈给恶心开门，恶心到家了。
